# C# Assignment Solutions

This folder contains solutions for four C# programming problems:

## 1. Cake World Application (CakeWorld.cs)
- **Purpose**: Online cake ordering system with validation and discount calculation
- **Features**: 
  - Custom exception handling for invalid flavours
  - Quantity validation
  - Discount calculation based on cake flavour
- **To Run**: `csc CakeWorld.cs && CakeWorld.exe`

## 2. Candy Craze Application (CandyCraze.cs)
- **Purpose**: Candy shop discount calculator
- **Features**:
  - Flavour validation
  - Discount calculation based on candy type
  - Object-oriented design with classes
- **To Run**: `csc CandyCraze.cs && CandyCraze.exe`

## 3. Library Management System (LibraryManagement.cs)
- **Purpose**: Book management with constructor concepts
- **Features**:
  - Default and parameterized constructors
  - Average pages calculation
  - Late fee calculation
- **To Run**: `csc LibraryManagement.cs && LibraryManagement.exe`

## 4. Hasina Cabs Application (HasinaCabs.cs)
- **Purpose**: Cab fare calculation system
- **Features**:
  - Booking ID validation with specific format
  - Inheritance implementation
  - Fare calculation based on cab type and distance
- **To Run**: `csc HasinaCabs.cs && HasinaCabs.exe`

## Key Concepts Covered
- Custom Exception Classes
- Object-Oriented Programming
- Inheritance
- Constructors (Default and Parameterized)
- Properties and Methods
- Input Validation
- Mathematical Calculations

## Sample Inputs and Outputs
Each application includes sample test cases as specified in the requirements. Run the applications and follow the prompts to test different scenarios.
